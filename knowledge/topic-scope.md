# 选题口径与排除规则

## 选题构成（双轨）

公众号定位仍以**中国公司相关并购**为主，但**国际重大并购**也纳入选题池。
每日清单建议**约 70% 中国 + 30% 国际**，灵活调整：当日中国侧素材偏少或国际侧出现重磅时可调换比例。

---

## 轨道 A：中国公司相关 M&A（主力）

### 纳入范围

✓ **A 股上市公司** 发起的并购 / 重组 / 要约收购  
✓ **港股上市公司**（中资）的收购或被收购  
✓ **中概股**（美股上市的中国企业）的 M&A  
✓ **中国民企** / **国企** 的重大收购（无论是否上市）  
✓ **跨境并购**：中国企业收购海外资产 OR 外资收购中国资产（需有合规/监管看点）  
✓ **PE / VC 相关**：中国 PE 退出、股权交易、控制权变更（≥5000 万 RMB）  
✓ **借壳 / 分拆 / 重组**：重大资产重组、借壳上市  

### 排除范围

❌ 纯股权激励 / 员工持股计划（非外部交易）  
❌ 定向增发（非实质性并购）  
❌ 股份回购（自家人买卖）  
❌ 行政合并（不涉及交易对价）  
❌ 金融衍生品 / 期货 / 期权（非实体资产）  
❌ 小额收购（<5000 万 RMB，除非有新闻价值）  
❌ 纯融资事件（融资轮次不算并购）  
❌ 海外中资企业之间的交易（对中国市场关联性弱）  

---

## 轨道 B：国际重大并购（补充）

### 纳入范围（满足任一条即可）

✓ **知名跨国公司**（Big Tech / 金融巨头 / 能源化工 / 医药等）发起或被收购，对价 ≥ 10 亿美元  
✓ **知名消费品牌**（奢侈品 / 快消 / 餐饮 / 家电 / 服装 / 美妆 / 汽车）易主、被分拆、私有化、跨界收购  
✓ **工业制造巨头**（半导体、汽车零部件、航空航天、能源装备、工业自动化等）的并购、整合、分拆  
✓ **行业格局变量级**交易：会改变某个行业全球竞争结构的（如 GPU / 云计算 / 创新药 / 矿业 / 媒体平台）  
✓ **中国读者熟悉的品牌**（苹果、特斯拉、LVMH、雀巢、宝洁、迪士尼、Netflix、波音、空客、台积电、三星、ASML、博通、辉瑞、罗氏、宝马、丰田、星巴克、麦当劳、可口可乐等）  
✓ **拥有中国业务/中国渠道**的国际公司，并购可能影响中国市场（如分销权、品牌授权变更）  

### 排除范围

❌ 小额财务投资（<10 亿美元，除非有强烈战略意义）  
❌ 与中国市场/读者无关、关注度低的纯本土化整合（如美国地方银行并购）  
❌ 信息源全是英文且无中文权威报道的（数据可得性 < 2 分）  
❌ 监管阶段、信息披露极少的小道消息  

### 国际题材的写作角度建议

为保持公众号差异化，国际题材要尽量切到"对中国读者有意义"的角度：
- 标的或买方在中国有业务/品牌，并购对中国市场的影响
- 与中国企业的对照（中国企业在做相同事/不同事）
- 全球产业链/供应链格局对中国玩家的影响
- 监管比较（美/欧反垄断 vs 中国市场监管总局经营者集中审查）

## 时间窗

### 默认
**过去 48 小时** — 最新、热度最高的选题

### 特例（可放宽至 3 周）
- 重大案例刚过会 / 刚披露（如科创板最大并购案）
- 行业群体事件（如有色出海潮）
- 政策驱动的新品类（如创新药出海 BD 爆发）

### 绝对冷线
超过 4 周的陈年案例，除非能挖出新的 follow-up 或行业启示，否则不选。

---

## 选题价值判断维度

### 热度（20%）
新闻搜索量、舆论提及度、股票交易量异常

### 独特性（20%）
是否有新的交易结构 / 新的参与方 / 新的赛道组合

### 可分析空间（30%）
- 定价是否合理？
- 动机是否隐晦？
- 结构是否有漏洞？
- 后续不确定性大不大？

### 争议 / 共鸣度（30%）
- 小股东利益冲突？
- 国资 vs 市场化定价？
- 大股东"肥水不外流"？
- 员工前景不确定？

---

## 一条好选题的特征

✓ 有**核心数字** —— 对价、估值、业绩承诺  
✓ 有**利益冲突** —— 不只是"好事件"  
✓ 有**故事线** —— 一句话能讲清楚  
✓ 有**开放式问题** —— 不是"我给出答案"，而是"你怎么看"  
✓ 有**可核实来源** —— 不是传言  

---

## 反编造护栏

- 【强制】选题清单里的数字，每一个都要标注来源链接
- 【强制】拿不准的标「待核实」，绝不预判
- 【强制】选定后在写作前回一手公告核实所有数字
- 【强制】任何无出处的数据在正文里必须写成"未披露"或"待核实"
