# 公众号排版模板（基准风格：爱普股份接盘挪亚圣诺）

## 文件命名规范

`YYYY-MM-DD-精简标题.md`

例：`2026-06-06-爱普股份接盘挪亚圣诺.md`

---

## 标准结构

```markdown
# 标题（问句 / 悬念 / 数字冲击，≤ 30 字）

[开场句 2：时间节点 / 另一个事实，制造对比，1-2 句]

[开场句 3：可选，交代背景或抛出核心问题，1-2 句]

---

[段落]
```

## 排版细则

### 标题层级
- `#` 文章大标题（只用一次，在最顶部）
- `###` 节标题（**不用 `##`**）
- 不用 `####` 或更深层级

### 开场（无小标题）
- 文章开头**不加任何小标题**，直接写 2-4 个短段落
- 每段 1-3 句，制造节奏感
- 第一句必须是具体事实或数字，**禁止用"近年来 / 随着 / 在当前背景下"起头**

### 节与节之间
- 每个 `###` 节前后都用 `---` 隔开
- 收尾段（无标题）前也加 `---`

### 段落长度
- 单段 **1-6 句**，约 50-150 字
- 同一节可以有 2-3 段，但每段保持独立节奏
- 关键判断单独成段，加粗

### 加粗规则
- 只加粗**核心判断句 / 金句**，不加粗普通数据
- 加粗句通常放在段末或独立成行
- 整篇文章加粗句不超过 6 处

### 数字格式
- 保留原始格式（4.5 亿元、11.5 倍、3897 万元）
- 数据后紧跟单位
- 无来源的数字标注「待核实」，**绝不估算**

### 收尾结构（固定格式）
```
---

[1-3 句回扣开头悬念，点出利益归属]

**[最终判断金句加粗]**

[开放问题 + "评论区聊聊"]

---

> **资料来源**
```

### 来源格式
```
> **资料来源**
> 1. 媒体名《标题》(YYYY-MM-DD) — https://...
> 2. 媒体名《标题》— https://...
>
> 注：文中 XX 数据均转引自上述报道对公告的披露。
```

## 禁止事项

| 禁止 | 原因 |
|---|---|
| `##` 作为节标题 | 本模板统一用 `###` |
| 开场加 `## 小标题` | 开场段落不加标题，直接入戏 |
| "近年来 / 随着 / 在当前 / 值得注意" | AI腔套话，参见 anti-cliche.md |
| 加粗普通数据 | 加粗只留给判断句 |
| 节与节之间不加 `---` | 公众号排版需要视觉分隔 |
| 来源块里的链接不写日期 | 尽量标注，增加可信度 |

```markdown
# 新华制药谈了14个月不要，爱普股份为何火速接盘？

新华制药谈了14个月，走了。

2026年2月6日，新华制药公告终止收购挪亚圣诺（太仓）。公告里只有一句话：双方未达成一致。14个月谈判，一句话收场。

同一天，爱普股份签了意向协议。

---

### 挪亚圣诺是什么？

这家公司不出名，但在鱼油原料这个细分市场里有点分量。

2024年，挪亚圣诺营收5.27亿元，净利润4824万元。2025年，营收掉到4.51亿元，净利润掉到3897万元。收入跌了14%，利润跌了19%，而且不是平稳下滑，是加速的。

**业绩连年下滑，前买家谈了超过一年放弃——这就是爱普股份花4.5亿接手的那个标的。**

---

### 溢价超200%，值吗？

（正文……）

---

回到开头的那个问题：**新华制药谈了14个月后放弃，那14个月里到底发现了什么？**

你觉得爱普这次是看到了别人没看到的机会，还是高估了自己消化这个资产的能力？评论区聊聊。
```

## 发布前检查清单

- [ ] 标题 ≤ 30 字，有钩子（问句 / 数字 / 反差）
- [ ] 开场无小标题，直接入戏
- [ ] 节标题全部用 `###`，节间全部有 `---`
- [ ] 加粗句 ≤ 6 处，均为判断金句
- [ ] 收尾有回扣开头 + 开放问题
- [ ] 所有数字有来源或标注「待核实」
- [ ] 无 AI 腔套话（对照 anti-cliche.md）
- [ ] 来源块格式正确，链接可访问
